# Privacy Policy for FC Tweak

Last updated: January 2025

## Overview

FC Tweak is a browser extension that helps you focus by blocking distracting elements on Farcaster.xyz. Your privacy is important to us, and this extension is designed with privacy as a core principle.

## Data Collection

**FC Tweak does not collect any personal data.**

Specifically, we do not collect, store, or transmit:
- Personal information
- Browsing history
- Usage statistics
- Analytics data
- Any other user data

## Data Storage

The extension stores only one piece of information locally in your browser:
- Your toggle preference (enabled/disabled state)

This data:
- Is stored locally using Chrome's `storage.sync` API
- Never leaves your browser (except to sync across your own Chrome devices if you're signed into Chrome)
- Can be deleted by uninstalling the extension

## Permissions

The extension requests the following permissions:

- **activeTab**: Required to modify the Farcaster.xyz page to hide feed content
- **storage**: Required to remember your toggle preference

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date above.
